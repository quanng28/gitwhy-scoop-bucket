# GitWhy Scoop Bucket

Scoop bucket for [GitWhy](https://gitwhy.ai) — the context layer for git.

## Install

```powershell
scoop bucket add gitwhy https://github.com/quanng28/gitwhy-scoop-bucket
scoop install git-why
```

## Usage

```powershell
git why setup      # Set up GitWhy in your repo
git why --version  # Verify installation
```
